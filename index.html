Cloud DevOps Project

Deploy a Static Website on AWS S3 with CI/CD via GitHub Actions

This project demonstrates how to host a static website using Amazon S3 and automate the deployment process with GitHub Actions. Whenever code is pushed to the repository, the CI/CD pipeline automatically builds and updates the website on S3, ensuring fast, reliable, and version-controlled deployments without manual intervention.
